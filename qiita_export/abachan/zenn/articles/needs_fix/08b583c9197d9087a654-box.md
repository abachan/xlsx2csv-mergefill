# はじめに
クラウドストレージサービス「Box」は、多くの企業や個人に利用されています。Windows環境でBoxのフォルダやファイルをエクスプローラから直接操作できると、操作性や業務効率が向上します。本記事では、Box Driveという公式ツールを使って、Boxのフォルダやファイルをエクスプローラから表示する方法について解説します。

# 手順

(1) [Boxのウェブサイト](https://www.box.com/resources/downloads)にアクセスし、Box Driveのインストールファイルをダウンロードします。
   
![image.png](https://qiita-image-store.s3.ap-northeast-1.amazonaws.com/0/3963503/d5d92115-eb80-b181-6b5f-ab110856cb87.png)

(2) ダウンロードが完了したら、インストーラを実行します。
(3) 「ログイン」ボタンを押下します。

![image.png](https://qiita-image-store.s3.ap-northeast-1.amazonaws.com/0/3963503/742d3fe4-6ab2-80e2-31d3-7ffcf618172f.png)

(4) Boxアカウントのメールアドレスを入力します。

![image.png](https://qiita-image-store.s3.ap-northeast-1.amazonaws.com/0/3963503/152dca2c-ef4c-6e57-0f1b-ffe9d977b6fa.png)

(5) パスワードを入力し、「ログイン」ボタンを押下します。

![image.png](https://qiita-image-store.s3.ap-northeast-1.amazonaws.com/0/3963503/e5c2a3a1-a7f0-0ffb-e60d-0eb03875e07c.png)

(6) 正常にログインできると、以下の画面が表示されます。

![image.png](https://qiita-image-store.s3.ap-northeast-1.amazonaws.com/0/3963503/7887a422-48ff-0a62-5822-9f38b35bfac2.png)

(7) しばらくすると、Box Driveが起動します。

![image.png](https://qiita-image-store.s3.ap-northeast-1.amazonaws.com/0/3963503/7a4371e4-8adc-3358-36b9-3f369a60e523.png)

(8) エクスプローラを開くと、Boxのクラウドフォルダがローカルドライブとして表示されます。
   - 通常、`Box`という名前のドライブとして表示されます。

![image.png](https://qiita-image-store.s3.ap-northeast-1.amazonaws.com/0/3963503/a726e6d6-e501-209f-313d-3c37af5e2a34.png)

### Box Driveのアイコンとファイルまたはフォルダのステータス
Box Driveのアイコンを見ると、ファイルとフォルダのステータスを一目で確認できます。
![image.png](https://qiita-image-store.s3.ap-northeast-1.amazonaws.com/0/3963503/9d5d20ac-d6b5-e62d-2626-ddc276262114.png)

### 注意点

- **インターネット接続**が必要です。Box Driveはクラウドベースで動作しているため、オフライン環境ではアクセスできません。
- セキュリティポリシーによって、Box Driveの使用が制約される場合があります。企業のITポリシーに従って使用してください。

### まとめ

Box Driveを利用することで、Windows環境でもBoxのクラウドストレージをエクスプローラから直感的に操作することができます。この記事の手順に従って、Boxのフォルダやファイルをエクスプローラに表示し、業務効率を向上させましょう。
