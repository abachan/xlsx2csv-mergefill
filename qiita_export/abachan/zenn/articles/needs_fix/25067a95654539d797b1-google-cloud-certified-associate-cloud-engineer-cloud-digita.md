## Associate Cloud Engineerとは？

**Google Cloud Certified - Associate Cloud Engineer**は、Google Cloud Platform（GCP）の基本的な運用や管理、デプロイメントに関する知識とスキルを証明する資格です。具体的には、以下の能力が求められます。

- GCP上でのアプリケーションのデプロイと管理
- クラウドソリューションのモニタリングと運用
- 基本的なセキュリティの確保
- GCPリソースの設定と管理

https://cloud.google.com/learn/certification/cloud-engineer?hl=ja

## Cloud Digital Leaderとの違い

以前取得したCloud Digital LeaderとAssociate Cloud Engineerの主な違いは、試験内容の深さと範囲にあります。

- **Cloud Digital Leader**: クラウドの基本概念やGCPのサービス概要、ビジネス価値に焦点を当てた内容。
- **Associate Cloud Engineer**: GCPの具体的なサービスや操作方法、実践的なスキルに重点を置いた内容。

そのため、Associate Cloud Engineerの方が技術的な知識や実務経験がより求められる試験と言えます。

## 学習計画の立て方

### 1. 学習期間の設定

Cloud Digital Leaderの経験を踏まえ、約2週間の学習期間を設定しました。週に20〜30時間の学習時間を確保し、計画的に進めました。

### 2. 学習内容の整理

以下のトピックを重点的に学習しました。

- GCPの主要サービス（Compute Engine、App Engine、Kubernetes Engineなど）
- IAM（Identity and Access Management）の設定と管理
- ネットワーキング（VPC、サブネット、ファイアウォールなど）
- ストレージサービス（Cloud Storage、BigQuery、Cloud SQLなど）
- モニタリングとロギング（Cloud Monitoring、Cloud Logging）

### 3. 試験前の最終確認
**Google Cloud Skill Boost**
Google Cloudが提供する公式の学習プラットフォームで、試験に向けたコースやハンズオンの受講が可能です。無料で利用できるため、最新の情報やサービスの詳細を学習するのに役立ちます。 


**Cloud Digital Leader公式模擬問題**
Googleが提供する公式の模擬問題集です。無料で受験可能で、本番と似た形式の問題に取り組むことができます。試験前に一度解いておくことをおすすめします。

## 試験当日の流れと注意点

### 1. 試験形式

- **試験時間**: 120分
- **問題数**: 50問
- **出題形式**: 単一選択および複数選択

### 2. 試験当日のポイント

- **時間配分**: 1問あたり約2分を目安に解答。難しい問題は一旦飛ばし、後で見直す戦略を取りました。
- **冷静さの維持**: 模擬試験での経験から、焦らず冷静に問題に取り組むことができました。
